# 网页版视频压缩工具

一个完全在浏览器中运行的视频压缩工具，支持精确控制输出文件大小，无需上传到服务器，保护用户隐私。

![HTML](https://img.shields.io/badge/HTML-100%25-blue)
![JavaScript](https://img.shields.io/badge/JavaScript-100%25-yellow)
![Client_Side](https://img.shields.io/badge/Client_Side-Only-green)

## 🌟 主要功能

- **完全本地处理**：所有压缩操作在浏览器中完成，视频不会上传到任何服务器
- **精确大小控制**：支持设置具体的目标文件大小（MB/KB）
- **多种压缩选项**：可调整视频质量、分辨率、帧率
- **实时预览**：压缩前后对比，直观显示压缩效果
- **智能推荐**：根据原始文件大小自动推荐合适的目标大小
- **拖拽上传**：支持拖拽文件到上传区域
- **响应式设计**：适配各种屏幕尺寸

## 📋 技术特性

- **纯前端实现**：使用HTML5 File API、Canvas API和Web Workers
- **无依赖**：不依赖任何外部库或框架
- **隐私保护**：所有处理都在客户端完成
- **跨平台**：支持所有现代浏览器
- **轻量级**：单HTML文件，无需安装

## 🚀 使用方法

### 基本使用

1. **上传视频**
   - 点击上传区域或拖拽视频文件到指定区域
   - 支持MP4、WebM、OGG格式，最大500MB

2. **设置压缩参数**
   - **目标文件大小**：输入具体数值并选择单位（MB/KB）
   - **预设大小**：快速选择2MB、5MB、10MB、20MB、50MB、100MB
   - **高级设置**：可调整视频质量、分辨率、帧率

3. **开始压缩**
   - 点击"开始压缩视频"按钮
   - 查看实时压缩进度

4. **下载结果**
   - 压缩完成后预览效果
   - 点击"下载压缩视频"保存到本地

### 高级功能

- **自动推荐**：系统会根据上传的视频大小智能推荐压缩目标
- **精确控制**：可以指定精确的文件大小，系统会尽量匹配
- **质量对比**：压缩前后文件大小对比，显示节省百分比

## 🛠️ 技术实现

### 核心技术

- **HTML5 File API**：处理文件上传和读取
- **Canvas API**：视频帧处理和编码
- **Video Element**：视频预览和元数据提取
- **Blob API**：生成和下载压缩文件
- **Web Workers**：后台处理视频压缩（模拟）

### 压缩算法

- 基于视频质量、分辨率和帧率的多参数调节
- 智能比特率控制以实现目标文件大小
- 保持视频质量的同时减少文件体积

## ⚠️ 注意事项

1. **性能考虑**：大视频文件（>100MB）压缩可能需要较长时间
2. **内存限制**：浏览器内存有限，超大文件可能导致浏览器卡顿
3. **格式支持**：输出格式与输入格式相同
4. **移动设备**：在移动设备上处理大文件可能较慢
5. **模拟压缩**：当前版本为演示版，实际项目中可集成FFmpeg.js实现真实压缩

## 🔧 开发说明

### 本地运行

1. 下载 `index.html` 文件
2. 在浏览器中直接打开即可使用

### 自定义修改

- **界面样式**：修改CSS部分调整外观
- **压缩参数**：调整JavaScript中的压缩算法参数
- **文件限制**：修改文件大小和格式限制
- **添加功能**：扩展JavaScript功能模块

### 真实压缩实现

当前版本为演示用途，模拟了压缩过程。如需实现真实压缩，可考虑：

1. 集成 [FFmpeg.js](https://github.com/Kagami/ffmpeg.js)
2. 使用 [Video.js](https://videojs.com/) 处理视频
3. 实现基于WebAssembly的编解码器
