# 🎬 网页版视频压缩工具

一个完全在浏览器中运行的视频压缩工具，支持精确控制输出文件大小，无需上传到服务器，最大程度保护用户隐私。

**🌐 在线体验**：[https://cjz-wr.github.io/video-compression/](https://cjz-wr.github.io/video-compression/)

**📅 文档更新**：2026年1月4日

## ✨ 核心特性

- **🔒 隐私优先**：所有操作均在本地浏览器中完成，视频文件**不会上传**至任何远程服务器。
- **🎯 精确控制**：支持通过滑块或直接输入，精确控制视频质量、分辨率、帧率和**目标文件大小**。
- **📊 实时对比**：提供压缩前后的视频预览、文件大小对比和节省空间百分比。
- **🖱️ 便捷交互**：支持文件拖拽上传，提供清晰的操作指引和视觉反馈。
- **📱 响应式设计**：适配桌面和移动设备，在各种屏幕尺寸上均能良好显示。

## 🚀 快速开始

1.  **访问应用**：在浏览器中打开 [项目网址](https://cjz-wr.github.io/video-compression/)。
2.  **上传视频**：点击上传区域或直接将视频文件拖入。
3.  **调整设置**：使用控制面板调整压缩参数（质量、分辨率或直接设置目标大小）。
4.  **开始压缩**：点击“开始压缩视频”按钮。
5.  **下载结果**：压缩完成后，预览效果并下载新视频。

## ⚙️ 功能详解

### 视频上传与预览
- **格式支持**：`MP4`、`WebM`、`OGG` 等常见格式。
- **文件信息**：上传后即时显示原始文件的名称、大小、时长和分辨率。
- **即时预览**：提供原始视频的播放控件。

### 智能压缩控制
- **视频质量**：低、中、高三档预设，平衡画质与体积。
- **分辨率调节**：支持从 480p、720p、1080p 到原始尺寸的多档调节。
- **目标文件大小**（特色功能）：可直接输入期望的输出文件大小（单位MB/KB），工具会智能计算参数以匹配目标。

### 进度与结果
- **进度展示**：实时显示压缩进度条。
- **效果对比**：清晰对比压缩前后的**文件大小**，并计算节省的百分比。
- **一键下载**：生成带有 `_compressed` 后缀的新文件。

## 🛠️ 技术实现

本项目为纯前端实现，利用现代浏览器技术提供完整的压缩流程体验：
- **核心技术**：HTML5 File API、Video Element、Canvas API、Blob API。
- **处理方式**：所有计算均在用户本地浏览器中执行，保障数据安全。
- **设计模式**：采用响应式布局和模块化交互设计。
